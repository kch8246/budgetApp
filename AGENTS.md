# AGENTS.md

## 프로젝트 설명
이 저장소는 CSV 기반 가계부 CLI 앱 프로젝트다.
CSV 파일을 입력으로 받아 거래 내역을 읽고, 수입/지출/잔액/카테고리별 결과를 CLI에서 다루는 것을 목표로 한다.

## 코딩 규칙
- 타입 힌트는 필수다.
- 함수 하나의 길이는 최대 50줄을 넘기지 않는다.

## TDD 규칙
- 구현 전에 반드시 테스트를 먼저 작성한다.
- 새 기능은 테스트 실패 확인, 구현, 리팩터링 순서로 진행한다.

## 품질 규칙
- 순환복잡도는 10 이하로 유지한다.
- 복잡도가 10을 넘는 변경은 분해해서 수정한다.

## 품질 검수 규칙
- 커밋 전에 반드시 `qa_engineer` 서브에이전트로 품질 검수를 수행한다.
- 검수 결과에 발견 사항이 있으면 먼저 해결한 뒤 커밋한다.

## 테스트 실행 명령어
- `pytest`
- `radon cc`

## 커밋 규칙
- 하나의 기능이 개발되면 즉시 커밋하고 푸시한다.
- 커밋은 기능 단위로 작게 나눈다.
